# プロジェクト管理システム

プロジェクト管理機能とルーティンジョブ工数管理を統合した包括的なWebアプリケーションです。

## 📋 主要機能

### 🚀 プロジェクト管理
- **プロジェクト作成・管理**: プロジェクトの作成、編集、進捗管理
- **タスク管理**: 各プロジェクトのタスクの詳細管理
- **進捗追跡**: プロジェクトとタスクの進捗状況を視覚的に表示
- **担当者アサイン**: タスクへの担当者の割り当て

### 📊 ガントチャート
- **タイムライン表示**: プロジェクトタスクの時系列表示
- **プロジェクト別フィルター**: 特定のプロジェクトのガントチャート表示
- **優先度別色分け**: タスクの優先度による視覚的な色分け
- **進捗率表示**: 各タスクの進捗率をバー上に表示

### ⏰ ルーティンジョブ工数管理
- **工数記録**: 日々のルーティン業務の工数入力・管理
- **記録一覧**: 工数記録の検索・フィルタリング機能
- **編集・削除**: 既存の工数記録の編集・削除機能
- **期間指定フィルター**: 日付範囲による記録の絞り込み

### 📈 統計・分析機能
- **プロジェクト統計**: プロジェクトの完了率、タスク統計
- **ルーティン統計**: 工数の集計、担当者別・ジョブ別分析
- **期間指定分析**: 特定期間の統計データ表示
- **効率性指標**: 予定工数と実績工数の比較分析

### ⚙️ マスタ管理
- **担当者管理**: 新しいメンバーの追加・管理
- **ルーティンジョブ管理**: 定期業務項目の追加・管理
- **プロジェクト作成**: 新規プロジェクトの作成

## 🛠 技術スタック

- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **バックエンド**: Node.js, Express.js
- **データ管理**: JSON ファイルベースの永続化
- **プロセス管理**: PM2
- **デザイン**: レスポンシブデザイン、グラスモルフィズム効果

## 🚀 インストールと起動

### 必要な環境
- Node.js (v14以上)
- npm

### セットアップ手順
```bash
# リポジトリのクローン
git clone [repository-url]
cd webapp

# 依存関係のインストール
npm install

# 通常の起動
npm start

# PM2での起動（推奨）
npm install pm2
npx pm2 start ecosystem.config.js

# PM2の状態確認
npx pm2 status

# ログ確認
npx pm2 logs project-management-system --nostream
```

### アクセス方法
ブラウザで `http://localhost:3000` にアクセスしてください。

## 📱 使用方法

### 1. プロジェクト管理
1. **「プロジェクト」タブ**でプロジェクト一覧を確認
2. **「プロジェクト追加」**ボタンで新規プロジェクトを作成
3. プロジェクトカードをクリックして詳細表示・タスク管理

### 2. ガントチャート
1. **「ガントチャート」タブ**でタスクのタイムライン表示
2. プロジェクトフィルターで特定プロジェクトに絞り込み
3. タスクの期間と進捗を視覚的に確認

### 3. ルーティンジョブ管理
1. **「ルーティンジョブ」タブ**で工数入力
2. 担当者、ジョブ、工数、日付を入力して記録
3. 記録一覧で過去の工数を確認・編集

### 4. 統計分析
1. **「統計」タブ**で各種統計を表示
2. プロジェクト統計/ルーティン統計を切り替え
3. 期間を指定して詳細分析

### 5. マスタ管理
1. **「管理」タブ**で各種マスタデータを管理
2. 担当者、ルーティンジョブ、プロジェクトを追加


## 🌐 API エンドポイント

### データ取得
- `GET /api/data` - 全データ取得
- `GET /api/projects` - プロジェクト一覧
- `GET /api/tasks` - タスク一覧
- `GET /api/members` - 担当者一覧
- `GET /api/routine-jobs` - ルーティンジョブ一覧
- `GET /api/routine-entries` - ルーティン工数記録一覧
- `GET /api/stats` - 統計データ

### データ操作
- `POST /api/projects` - プロジェクト追加
- `PUT /api/projects/:id` - プロジェクト更新
- `DELETE /api/projects/:id` - プロジェクト削除
- `POST /api/tasks` - タスク追加
- `PUT /api/tasks/:id` - タスク更新
- `DELETE /api/tasks/:id` - タスク削除
- `POST /api/members` - 担当者追加
- `POST /api/routine-jobs` - ルーティンジョブ追加
- `POST /api/routine-entries` - ルーティン工数記録追加
- `PUT /api/routine-entries/:id` - ルーティン工数記録更新
- `DELETE /api/routine-entries/:id` - ルーティン工数記録削除
